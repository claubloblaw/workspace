# Marcel Gruber Portfolio

Personal portfolio website for Marcel Gruber - Enterprise Web & AI Systems Architect.

## About

Single-page portfolio site built with Next.js 16, TypeScript, and Tailwind CSS. Minimal design with dark theme, inspired by narrative-driven personal sites but with distinct sections for services and content.

## Features

- **Dark theme** with purple accent colors
- **Smooth scroll** navigation between sections
- **Responsive design** - mobile-first approach
- **Sections**:
  - Hero with compelling tagline and professional photo
  - About/Story - authentic, builder-focused narrative
  - Services - Enterprise Web, WordPress, AI & Agentic Systems, SEO
  - Blog link to Sitecore blog
  - Contact information

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Inter font from Google Fonts
- Deployed static/SSG

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

The site will be available at `http://localhost:3000`
