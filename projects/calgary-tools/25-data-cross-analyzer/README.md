# Calgary Data Cross-Analyzer

The meta-tool: analyzes relationships between multiple datasets to find hidden correlations.

## What It Does

Pulls data from 5+ different datasets and looks for interesting correlations:
- Building permits vs property values
- Crime rates vs business density
- 311 complaints vs demographics
- Park access vs property values

## Why It's Interesting

Most insights come from combining datasets, not analyzing them in isolation. This tool automates the discovery of cross-dataset patterns.

## Output

- `correlations.json` - Statistical correlations found
- `insights.html` - Interactive dashboard
- `recommendations.txt` - Investment/policy recommendations
