# Neighborhood Gentrification Index

Identifies neighborhoods experiencing gentrification based on multiple signals.

## Signals Tracked

- Rising property values
- Increasing building permits
- Changing business types (artisanal vs chain)
- Demographic shifts
- Decreasing crime

## Output

- `gentrification_scores.json`
- `gentrification_map.html`
- `early_stage_areas.csv` - Areas just starting to gentrify
